POST /api/attempts/:testId/start

Start a new exam attempt (requires SEB headers if configured).

Headers

Authorization: Bearer <your_jwt_token>
X-SafeExamBrowser: <hashed_config_key> (if SEB enabled)

Example Response

{
  "attemptId": "xyz789",
  "questions": [...],
  "timeLeft": 30
}

POST /api/attempts/:attemptId/submit

Submit exam answers.

Request Body

{
  "answers": [
    { "questionId": "q1", "selectedOption": "A" },
    { "questionId": "q2", "selectedOption": "B" }
  ]
}

Example Response

{
  "score": 85,
  "status": "passed"
}
